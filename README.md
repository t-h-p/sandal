# Sandal
A tool to graph your filesystem on Unix-like systems.
Successfully built on MacOS and OpenSUSE in the past.
Example output:
![Graph of source tree](./output.png)
Usage: 
```
git clone https://github.com/t-h-p/sandal.git
cd sandal
cabal run --{TARGET DIRECTORY} -o {OUTPUT FILE}

``` to build and try.
